**[NOVEL FRAMEWORK]:**  
For a 3SAT instance $\phi$ on $n$ variables, define the **partial assignment complex** $X_\phi$ as follows. Vertices are partial assignments $v \in \{0,1,\ast\}^n$ that do not falsify any clause of $\phi$. There is a directed edge $u \to v$ if $u \prec v$ (i.e., $v$ extends $u$ by assigning one $\ast$ to $0$ or $1$) and $v$ remains non-violating. Higher-dimensional cells are added whenever all boundary edges are present, making $X_\phi$ a simplicial complex. The **Satisfying Fiber** $\mathcal{F}_\phi$ is the set of full assignments (no $\ast$) in $X_\phi$.

At the critical shattering density $\alpha \approx 4.267$, $X_\phi$ is known to be a **hyperbolic** space with exponentially many "dead-end" components: partial assignments that cannot be extended to a solution, which act as topological obstructions. We equip $X_\phi$ with a non-trivial local coefficient system: assign to each directed edge a "charge" in $\mathbb{Z}$ that measures the *algorithmic cost* of that assignment extension. Specifically, for each edge $e = (u \to v)$, let:
$$ \omega(e) = K(v \mid \phi) - K(u \mid \phi) \pmod{2^{\Omega(n)}}, $$
where $K(\cdot \mid \phi)$ is the unbounded prefix-free Kolmogorov complexity conditioned on the full description of $\phi$. This defines a 1-cochain $\omega \in C^1(X_\phi; \mathbb{Z}_{2^{\Omega(n)}})$. Since $\omega = \delta K$ (the coboundary of the 0-cochain $v \mapsto K(v \mid \phi)$), it is a **cocycle**. For any path $\gamma$ from the empty assignment $\emptyset$ to a solution $s \in \mathcal{F}_\phi$, the **winding number** of $\gamma$ is defined as the sum of $\omega(e)$ over edges of $\gamma$ modulo $2^{\Omega(n)}$, which is exactly $K(s \mid \phi) - K(\emptyset \mid \phi) \equiv K(s \mid \phi)$ (since $K(\emptyset|\phi)=O(1)$).

The crucial property: at the shattering threshold, the space of satisfying assignments is a **Replica Symmetry Breaking (RSB) glass** with $2^{\Omega(n)}$ clusters that are algorithmically independent conditional on $\phi$. Consequently, for a random $\phi$, the Kolmogorov complexity of **any** satisfying assignment $s$ satisfies:
$$ K(s \mid \phi) = \Omega(n) $$
with probability $1-o(1)$. This is a consequence of the fact that each cluster core requires $\Omega(n)$ bits to specify within the set of all clusters, and any specific solution contains the same algorithmic information as its cluster core (up to $O(n)$ bits for intra-cluster variations).

**[PROPOSED STEP 1]:**  
**Lemma (Exponential Winding Number Lower Bound).**  
Let $\phi$ be a random 3SAT instance at the critical density. Then for **any** continuous path $\gamma$ in $X_\phi$ from $\emptyset$ to any $s \in \mathcal{F}_\phi$, the winding number $W(\gamma) := \sum_{e \in \gamma} \omega(e)$ satisfies:
$$ W(\gamma) \ge 2^{\Omega(n)} \quad \text{with high probability over } \phi. $$

*Proof sketch.* By definition, $W(\gamma) \equiv K(s \mid \phi) \pmod{2^{\Omega(n)}}$. Since $K(s \mid \phi) \ge c n$ for some constant $c>0$ (due to the algorithmic independence of clusters and the exponential number thereof), the residue class $W(\gamma)$ in $\mathbb{Z}_{2^{\Omega(n)}}$ is non-zero and in fact has magnitude at least $2^{c n}$. The winding number counts how many times the path "encircles" the algorithmic complexity obstructions; each dead-end region forces the path to accumulate additional winding because the only way to extend a partial assignment past a dead end is to backtrack and take a different branch, which corresponds to a jump in conditional complexity. A more topological justification: $X_\phi$ has a fundamental group whose abelianization has rank exponential in $n$, generated by loops around dead-end components. The cocycle $\omega$ pairs non-trivially with these generators, and any path to a solution must have a winding vector in $\mathbb{Z}^{2^{\Omega(n)}}$ of large norm. The exponential lower bound on $K(s|\phi)$ then translates to the required winding number modulo $2^{\Omega(n)}$ being exponentially large. ∎

Now consider a polynomial-time deterministic algorithm $\mathcal{A}$ that, given $\phi$, outputs a satisfying assignment. Its execution trace defines a path in $X_\phi$: the sequence of partial assignments corresponding to the algorithm's current knowledge about the solution (e.g., assignments to variables that have been fixed). At each step, the algorithm can only increase the winding number by an amount proportional to the algorithmic information acquired in that step, which is at most $O(\log n)$ (the size of the state). Over $\text{poly}(n)$ steps, the total winding number accumulated is $O(\text{poly}(n)\log n)$, which is less than $2^{\Omega(n)}$ for large $n$. Contradiction. Hence no polynomial-time algorithm exists.

**[BARRIER EVASION CHECK]:**  
- **Natural Proofs:** The invariant $W(\gamma)$ relies on the unbounded Kolmogorov complexity $K(v \mid \phi)$, which is **uncomputable** in polynomial time. The lower bound $K(s \mid \phi) = \Omega(n)$ is derived from the shattered cluster structure and algorithmic independence, but there is no efficient predicate that identifies such hard instances. The argument is non-constructive and avoids the Razborov–Rudich barrier.
- **Relativization:** The winding number argument depends on the local, sequential nature of computation. An oracle machine could, in a single step, query an oracle that returns a complete satisfying assignment, effectively jumping from $\emptyset$ to $s$ in one step without accumulating intermediate winding. Thus the bound does not hold in relativized worlds, evading the relativization barrier.
- **Algebrization:** No arithmetization or algebraic oracle extensions are used. The cohomological cocycle is defined purely topologically, and the analysis relies on algorithmic information theory and the geometry of the partial assignment space at the phase transition. Hence algebrization does not apply.
